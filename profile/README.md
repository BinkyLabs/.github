### OpenAPI ecosystem

- [OpenAPI Overlays Dotnet](https://github.com/BinkyLabs/openapi-overlays-dotnet) A CLI/GitHub Action/Docker image/Dotnet implementation of the OpenAPI Overlay specification. Allows you to parse, serialize and apply Overlay documents to OpenAPI documents. Integrated with [Microsoft.OpenAPI](https://www.nuget.org/packages/Microsoft.OpenApi).
- [OpenAPI Arazzo Dotnet](https://github.com/BinkyLabs/openapi-arazzo-dotnet). A dotnet implementation of the OpenAPI Arazzo specification. Allows you to parse and serialize Azrazzo documents. Integrated with [Microsoft.OpenAPI](https://www.nuget.org/packages/Microsoft.OpenApi). Work in progress/still private.

### TypeSpec

- [TypeSpec Muzzle](https://github.com/BinkyLabs/typespec-muzzle). A TypeScript/CLI that allows you to suppress validation rules infringements in TypeSpec definitions. Helpful to baseline definitions when introducing new rules.
- [TypeSpec Kiota emitter](https://github.com/BinkyLabs/kiota-typespec-emitter). A TypeSpec emitter to generate API clients using [kiota](https://aka.ms/kiota). Provides a richer alternative to TypeSpec emitters that is supported for production!
 
### Misc

- [OpenAI analyzers](https://github.com/BinkyLabs/openai-analyzers). Dotnet analyzers packed with best practices for security and performance when integrating with OpenAI APIs.
